# Mac快捷键

## 1. 系统

https://support.apple.com/zh-cn/HT201236

关闭窗口：Command + W

关闭应用：Command + Q

同一个应用内切换：Command + `

剪切文件：Command + C（复制），Option + Command + V（剪切）

## 2. 软件

### 2.1 Chrome

下一个Tab: Control + Tab

前一个Tab: Control + Shift + Tab

打开上次关闭的Tab：Shift + Commadn + T
